# Code

## Overview

This phase contains two notebooks with the same overall structure.

The analysis explores the effect of **4 different label aligning methods** and **2 window lengths** on the final results, producing 8 groups in total.

## Notebooks

### `Phase3_HMM_code (with 4 align methods).ipynb`
- Contains **1 selected group** (one label aligning method + one window length)
- Includes detailed **analysis, limitations, and potential improvements**

### `Phase3_HMM_code (All 8 groups).ipynb`
- Contains results for **all 8 groups**
- Useful for comparing the effect of different label aligning methods and window lengths
