# Results (csv)

## Overview

This folder contains HMM regime detection results in CSV format, serving as input files for Phase 4 backtesting.

Results are generated under **4 label aligning methods** (aligndef1–4) × **2 window lengths** (3 years / 5 years) = **8 configurations** in total.

## File Description

### Individual Configuration Results
Generated by `Phase3_HMM_code (with 4 align methods).ipynb` — one file per configuration:

| File | Label Align Method | Window Length |
|---|---|---|
| `HMM_aligndef1_3years.csv` | aligndef1 | 3 years |
| `HMM_aligndef1_5years.csv` | aligndef1 | 5 years |
| `HMM_aligndef2_3years.csv` | aligndef2 | 3 years |
| `HMM_aligndef2_5years.csv` | aligndef2 | 5 years |
| `HMM_aligndef3_3years.csv` | aligndef3 | 3 years |
| `HMM_aligndef3_5years.csv` | aligndef3 | 5 years |
| `HMM_aligndef4_3years.csv` | aligndef4 | 3 years |
| `HMM_aligndef4_5years.csv` | aligndef4 | 5 years |

### Summary File
| File | Description |
|---|---|
| `HMM_all8configs.csv` | Consolidates results from all 8 configurations above, generated by `Phase3_HMM_code (All 8 groups).ipynb` |

## Usage for Phase 4
The 8 individual CSV files are used as input for Phase 4 strategy backtesting.
